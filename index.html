Game Rules

  • The game has 3 modes: Easy, Hard, and Ultimate
  • You are to guess the correct number between 1 to 50
  • You have a limited number of attempts in each of the 3 modes: Easy Mode - 20 attempts Hard Mode - 15 attempts Ultimate Mode - 10 attempts
  • The random number is hidden behind the question mark (?) on the game page
  • The number of attempts is your score at the start of the game
  • You lose a point for every wrong answer you make
  • You lose the game when your score gets to 0
  • You win the game when you guess the correct number
  • You can click Play Again button to play a new game and try to beat your high score
